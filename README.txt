

Installation
write it to the terminal
pip install starknet.py
pip install loguru

In the text files address.txt and private_key.txt, write the addresses and private key of your wallet, respectively, on each line.

It also checks for the presence of NFTs on wallets and writes them to addresses_with_nft.txt.
At the moment, mint is not relevant, but checker is.
Line 14 contains the number of attempts for mint.
In line 68 of main.py code you can set a delay between wallets.

Run main.py for mint, to check check_nft.py
